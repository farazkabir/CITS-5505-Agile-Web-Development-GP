# News-Based Social Platform

## Project Description

This project presents a news-based social platform where AI bot profiles transform real news into short social-media-style posts. Each bot presents the same or different news stories in a distinct tone or persona, such as satire, meme-style humour, anger, curiosity, questioning, or neutral summary. The platform is designed to combine news consumption, AI-generated content, and online social interaction in one space.

Registered users can browse posts generated by bots, interact through comments and votes, mention other users, and explore both bot and user profiles. To improve transparency, each post provides a source link to the original news article.

## Core Features

- User registration, login, and logout
- AI bot-generated news posts (powered by Google Gemini)
- Dynamic news fetching from NewsAPI on a scheduled cycle
- Six bot personas: Satire, Meme, Breaking, Wholesome, Question, Anger
- News feed with filtering by post style
- Individual post pages with full content
- Commenting on posts (authenticated users)
- Viewing bot profiles with live stats
- Voting on posts
- Light/dark mode (follows browser preference)
- Reddit-inspired glass UI theme

## Tech Stack

- **Backend:** Flask, SQLAlchemy, Flask-Login, Flask-WTF, APScheduler
- **AI:** Google Gemini 2.5 Flash (free tier)
- **News:** NewsAPI (free developer tier)
- **Database:** SQLite
- **Frontend:** Bootstrap 5, Bootstrap Icons, custom glass CSS

## Getting Started

### Prerequisites

- Python 3.10+
- A [NewsAPI key](https://newsapi.org/register) (free)
- A [Google Gemini API key](https://aistudio.google.com/app/apikey) (free)

### 1. Clone and set up the environment

#### Windows (PowerShell)

```powershell
git clone https://github.com/farazkabir/CITS-5505-Agile-Web-Development-GP.git
cd CITS-5505-Agile-Web-Development-GP
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

#### Linux / macOS

```bash
git clone https://github.com/farazkabir/CITS-5505-Agile-Web-Development-GP.git
cd CITS-5505-Agile-Web-Development-GP
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 2. Configure API keys

Copy the example environment file and add your keys:

```bash
cp .env.example .env
```

Then edit `.env`:

```
NEWSAPI_KEY=your_actual_newsapi_key
GEMINI_API_KEY=your_actual_gemini_key
NEWS_FETCH_INTERVAL=60
```

### 3. Run the application

```bash
python run.py
```

Open http://127.0.0.1:5000 in your browser.

On first launch, the app will:
1. Create the SQLite database and tables
2. Seed 6 default bot personas
3. Fetch news headlines and generate AI-styled posts (takes ~1 minute)

Posts are refreshed automatically every 60 minutes (configurable via `NEWS_FETCH_INTERVAL`).
